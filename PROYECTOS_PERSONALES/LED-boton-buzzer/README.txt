Resumen del Proyecto

Este proyecto utiliza una placa Arduino para controlar tres LEDs y un buzzer mediante tres botones pulsadores. Cada botón alterna el estado de un LED específico (encendido o apagado). Además, el buzzer se activa cuando el LED1 está apagado y los LEDs 2 y 3 están encendidos al mismo tiempo.

Componentes necesarios:  
- 3 LEDs con sus resistencias correspondientes.  
- 3 botones con resistencias pull-up internas configuradas en el Arduino.  
- 1 buzzer.  
- 1 placa Arduino.  
- Protoboard y cables de conexión.  

Funcionamiento:  
1. Control de LEDs:  
   Cada botón alterna el estado de un LED:  
   - BOT1 controla LED1.  
   - BOT2 controla LED2.  
   - BOT3 controla LED3.  

2. Condición del buzzer:  
   El buzzer se activa únicamente si:  
   - LED1 está apagado.  
   - LED2 y LED3 están encendidos.  

Este proyecto es ideal para aprender conceptos básicos de electrónica y programación con Arduino, como el manejo de botones, LEDs, condiciones lógicas y salidas digitales.
